What is Array?

An array is a container object that holds a fixed number of values of a single type the length of an array is established when the array is created'(Arrays - Oracle Documentation') Every time that a programmer want to create something with more than one value, let say for an example a list of vegetables, a 'to-do' list, he can use an array. Different than other function on the Ruby language, array is specifically intended to expect argument whichever they are, a number, a word or a sentence. Let's take a sentence for an example, a sentence is made of different word put together ( excuse me for not going in the languages rules with verb and so on ..) We can use that sentence and brake it to different words, we can can compare this word to a word that we are looking for in our database and compare this word, and we can uses this knowledge to understand better the user and it s need. To summarize, array is not a single number, or a single word, it is a list that can contain many arguments inside of it.

What is Hash?

A hash function is any function that can be used to map data of arbitrary size to data of fixed size..the values returned by a hash function are called hash values, hash codes, hash sums, or simply hashes'(Hash function - Wikipedia, the free encyclopedia). If earlier we talked about 'what i an array', hashes is very similar to an array, but beside just creating a list, you have the option to sort it, map it, and give keys and order to the different list items. Imagine a long list of big company like google. Lets say that you are looking in the directory to find someone, How could you find him if all the thousand of people would be just laid on a simple list? Exactly because of his example ( and many other of course ) you need the option to map information.
